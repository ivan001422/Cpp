## Объявление и определение

```cpp
возвращаемый_тип имя_функции(список_параметров)
{
    // тело функции
}
```

Если реализация функции находится **после** `main()`, необходимо предварительно добавить **прототип** (объявление).  
Функция, определённая **до** `main()`, служит собственным прототипом.

```cpp
#include <iostream>

int add(int a, int b);   // прототип

int main()
{
    std::cout << add(2, 3);
    return 0;
}

int add(int a, int b)    // определение
{
    return a + b;
}
```

## Параметры и аргументы

- **Параметры** – переменные, перечисленные при объявлении функции.
- **Аргументы** – конкретные значения, передаваемые при вызове.

Функции, принимающие параметры, называются **параметризованными**.

```cpp
void printSum(int x, int y)   // x, y – параметры
{
    std::cout << x + y;
}

int main()
{
    printSum(5, 10);          // 5 и 10 – аргументы
}
```

## Возврат значений

### Тип `void`
Если функция ничего не возвращает, используется тип `void`.  
`return;` (без значения) прерывает выполнение и передаёт управление обратно.

```cpp
void sayHello()
{
    std::cout << "Hello!\n";
    return;          // необязателен, выход произойдёт автоматически
}
```

### Возврат значения
Когда тип отличен от `void`, инструкция `return` обязана вернуть значение **совместимого типа**.

```cpp
int getTen()
{
    return 10;
}

int main()
{
    int a = getTen();            // a = 10
    std::cout << getTen();       // 10
    std::cout << 10 + getTen();  // 20
}
```

> Несоответствие типов в `return` вызывает ошибку компиляции.

## Области видимости

### Локальная область
Создаётся любым блоком `{ }`. Переменные, объявленные внутри, называются локальными (функции, циклы, условия).

```cpp
void func()
{
    int val = 10;   // локальная переменная
}

int main()
{
    int val = 5;    // другая переменная
    func();
    // здесь val всё ещё 5
}
```

**Сокрытие имён** – внутренняя переменная временно перекрывает внешнюю в своей области видимости.  
Параметры функции также существуют только внутри её тела.

```cpp
void test()
{
    int i = 10;
    std::cout << i;   // 10
}

int main()
{
    int i = 5;
    test();           // выведет 10
}
```

### Глобальная область
Переменные, объявленные **вне всех функций** (в том числе вне `main()`), доступны из любого места программы, если не перекрыты локальными переменными.

```cpp
double pi = 3.1415;

int main()
{
    std::cout << pi;
}
```

## Передача указателей в функцию

Чтобы функция могла изменить внешнюю переменную, передаётся **указатель** на неё.

```cpp
void setToZero(int *ptr)
{
    *ptr = 0;   // изменение значения по адресу
}

int main()
{
    int x = 42;
    setToZero(&x);
    // x == 0
}
```

> **Рекомендация по стилю:** звездочку `*` прижимают к имени параметра (`int *ptr`),  
> потому что запись `int* a, b` создаёт только один указатель `a`, а `b` остаётся обычной переменной.

## Передача массивов в функцию

Массив можно передать тремя эквивалентными способами:

```cpp
void print1(int arr[10]);   // с указанием размера
void print2(int arr[]);     // без размера
void print3(int *arr);      // через указатель
```

Во всех случаях в функцию попадает лишь указатель на первый элемент.  
Поэтому длину массива часто передают отдельным параметром:

```cpp
void printArray(int arr[], int size)
{
    for (int i = 0; i < size; ++i)
        std::cout << arr[i] << ' ';
}
```
